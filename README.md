Abogadazo
================

[![Deploy to Heroku](https://www.herokucdn.com/deploy/button.png)](https://heroku.com/deploy)

Setup
-------------

This application requires:

- Ruby 2.2.2
- Rails 4.2.4

```
bundle install
```

Setup vars on:
```
database.yml & secrets.yml
```

```
rake db:create
```

```
rake db:smigrate
```

```
rake db:seed
```

Run server:
```
rails s
```
